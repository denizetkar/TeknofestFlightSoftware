# TeknofestFlightSoftware
Teknofest Roket Yarışması uçuş bilgisayarı kodudur. Arduino Uno, MPU9250 ve NEO 6M kullanılmıştır.
## Bağlantı Şeması
UNO Ground -> MPU9250 Ground, NEO 6M Ground  
UNO 5V     -> MPU9250 Vcc, NEO 6M Vcc  
UNO A4     -> MPU9250 SDA  
UNO A5     -> MPU9250 SCL  
UNO PIN3   -> NEO 6M RX  
UNO PIN4   -> NEO 6M TX
